# Sistema FO - v1.0.1

Sistema de gestão de Fatos Observados desenvolvido em Flask.

## Funcionalidades

- Login seguro com hash
- Controle de permissões por perfil
- Lançamento de FO
- Homologação
- Ranking
- Exportação BI
- Administração de militares
- Administração de usuários
- Administração de tipos de FO
